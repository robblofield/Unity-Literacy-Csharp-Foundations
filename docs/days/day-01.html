Day 1

Print Your First Message

Debugging Console Fundamentals Dickhole
Aim

Kick off the challenge by getting Unity to print a message into the Console. This establishes the foundation for observing and understanding everything your scripts do.

Objectives / Pseudocode Breakdown
  • 1. Create a new C# script and attach it to an empty GameObject in the scene.
  • 2. Inside Start(), print a custom message.
    • Use a sentence or your own name.
  • 3. Add a second message inside Update() and compare the behaviour.
Hints
  • Hint 1: The method you need is Debug.Log("your message here");
  • Hint 2: Remember: Start() runs once, Update() runs every frame.
Did I Get It Right?
  • When you enter Play mode:
    • You should see one message printed from Start().
    • You should see many repeated messages from Update() every frame.
  • Stopping Play mode should stop the console spam.
